Cursor OTEL Hook Installer

This package installs the OpenTelemetry integration for Cursor IDE, enabling observability of AI agent activities.

What will be installed:

  • Cursor OTEL Hook executable
  • Configuration templates
  • Per-user setup automation

Installation locations:

System files: /Library/Application Support/CursorOtelHook/

User hooks: ~/.cursor/hooks/

After installation:

  • Configuration is deployed to each user on login
  • Restart Cursor IDE for hooks to take effect
  • Traces will be sent to the configured OTEL endpoint
